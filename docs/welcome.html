Echo JS SDK 3.0

Echo offers a collection of powerful Cloud Services that do the heavy lifting for real-time, social applications. This SDK is designed to make it easy to consume these services using a JS API, Controls and Plugins. In addition the SDK helps developers build more consistent and interoperable JavaScript apps by standardizing the most common aspects of apps development, such as RESTful API handling, app configuration, dependancy loading, extensibility via plugins, general app deployment, etc. (Learn more).
